# Data releases

Every export run appends an entry here (generated by `scripts/export.mjs`). Full line-by-line history is in git.

## 2026-07-03 — schema v2

- **Schema upgrade** (same 3,268 wallets, 55 active brands — no address changes):
  - Per-wallet **evidence fields**: `evidence_type`, `evidence_source`, `evidence_url`, class-based `confidence_score`, `first_seen_at`, `as_of`
  - **Seed vs derived separation**: `seed_wallets` (1,234 — direct public evidence) and `derived_wallets` (2,034 — common-input-ownership expansion) are now distinct arrays with `is_seed_wallet` / `cluster_method`
  - **`volumeSuspectReasons`**: machine-readable why behind every under-review flag. Current: Rollbit + Gemini → `abnormal_avg_transfer_size`; Rain.gg → `manually_flagged_wash_or_treasury_pattern`; Gamba cleared from under-review this release
  - `wallet_role` added to the schema as `null` (role inference not yet shipped — never guessed)
- New file: `data/excluded-wallets.json` — the infra denylist, published so exclusions are auditable
- New docs: `DATA_DICTIONARY.md` (field-level definitions), this changelog
- New: `scripts/export.mjs` — the full pipeline, reproducible by anyone from the public API
- Site API additions: `GET /api/labels/export` (per-wallet provenance), `volumeSuspectReasons` on `/api/brands`
